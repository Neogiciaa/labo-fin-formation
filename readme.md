# Web API - Express + Sequelize(GoodBye Mongo) !

## Mise en place
Installer les dépendences avec la commande npm i
Créer le fichier .env sur base du fichier .env.example

## Commandes
Lancer la production
npm start

Lancer le mode dev (via Nodemon)
npm run dev
